Personalised Workout Generator

A Python CLI tool that generates personalized workout routines.

This is a console-based Workout Generator built using Python. The tool collects user information such as weight, height, goal weight, fitness level, workout intensity, available days, workout time, and calorie intake to create a tailored weekly workout routine. It supports beginner, intermediate, and advanced users with exercise lists customized for each level.

How it Works: The program prompts users for their physical measurements and preferences, including fitness level (Beginner, Intermediate, Advanced), workout intensity, days per week (1-7), session time, and daily calorie intake. It then generates a weekly routine by randomly selecting exercises from a level-specific list and assigning durations that fit within the user’s available time. The routine is adjusted to ensure the total time matches the user’s input, providing a balanced and personalized plan.

Rating: 9/10 – An excellent tool for personalized fitness planning with robust input validation, though it could benefit from a graphical interface or integration with fitness tracking apps.
