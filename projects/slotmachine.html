Slot Machine Game

A Python-based console slot machine game with betting and payouts.

This is a console-based Slot Machine game built using Python. The game simulates a 3x3 slot machine grid where players can deposit money, choose to bet on 1 to 3 lines, and place a bet amount. Winnings are calculated based on matching symbols across the bet lines, with higher payouts for rare combinations like three '7' symbols.

How it Works: Players start by depositing an amount, selecting 1 to 3 betting lines, and specifying a bet amount. The game generates a 3x3 grid of random symbols (A, B, C, D, E, F, or 7). Winnings are determined by the number of matching symbols on the selected lines, with multipliers applied based on the combination (e.g., three '7's yield higher payouts). The balance is updated after each spin, and players can choose to continue or cash out.

Rating: 7.5/10 – A fun and engaging implementation of a slot machine with solid betting mechanics, though a graphical interface could enhance the experience.
