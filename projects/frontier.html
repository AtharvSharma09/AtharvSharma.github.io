Markowitz's Efficient Frontier

A tool that generates optimized portfolio weights based on user-defined risk constraints.

This is a command-line tool built in Python that optimizes portfolio weights for NIFTY 500 stocks to maximize returns for a user-specified maximum annualized risk. It uses historical stock data from Yahoo Finance and applies the Efficient Frontier concept to determine optimal allocations.

How it Works: The tool downloads historical stock prices, calculates daily returns, and allows users to input a maximum risk level (e.g., 20%). It then uses the `scipy.optimize` library to compute the portfolio weights that maximize expected returns while keeping risk below the specified threshold. Users can provide custom weights via a CSV file, and the tool calculates portfolio risk using the annualized covariance matrix.

Rating: 9.5/10 – A robust tool for portfolio optimization with practical applications, though it could benefit from a graphical interface or real-time data integration.
