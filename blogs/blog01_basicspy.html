Python Basics: A Comprehensive Guide

What is Python?

Python is a high-level, interpreted programming language celebrated for its clean syntax and versatility. Created by Guido van Rossum in 1991, Python emphasizes code readability, allowing developers to write concise programs. Its applications span web development (Django, Flask), data science (Pandas, NumPy), automation, machine learning (TensorFlow, Scikit-learn), and more. Python’s extensive standard library and active community make it a top choice for beginners and professionals alike.

Installing Python

To start coding in Python, you need to install it on your system:

  1. Visit python.org and download the latest version (e.g., Python 3.12) for your operating system (Windows, macOS, or Linux).
  2. Run the installer. On Windows, ensure you check “Add Python to PATH” to make Python accessible from the command line.
  3. Open a terminal (Command Prompt on Windows, Terminal on macOS/Linux) and type python --version. If you see the version number, Python is installed correctly.
Tip: Use an IDE like VS Code or PyCharm for a better coding experience, with features like code completion and debugging.

Your First Python Program

Let’s write a classic “Hello, World!” program to get started: 

print("Hello, World!")

How to run it:
1. Create a file named hello.py.
2. Copy the code above into the file.
3. Open a terminal, navigate to the file’s directory, and run python hello.py.
Output: Hello, World!
The print() function displays text on the console. You can also print numbers or variables, making it a versatile tool for debugging and output.

Variables and Data Types

Variables are used to store data, and Python is dynamically typed, meaning you don’t need to declare the type explicitly. Here’s an example: 


name = "Rahul"    # String: Text data
age = 25          # Integer: Whole numbers
weight = 70.5     # Float: Decimal numbers
is_student = True # Boolean: True or False

You can check a variable’s type using the type() function: 


print(type(name))  # Output: 
print(type(age))   # Output:

Python’s flexibility with data types allows seamless transitions between operations, but be cautious with type mismatches (e.g., adding a string to an integer).

Basic Operations

Python supports arithmetic and string operations: 


# Arithmetic Operations
a = 10
b = 5
print(a + b)  # Addition: 15
print(a - b)  # Subtraction: 5
print(a * b)  # Multiplication: 50
print(a / b)  # Division: 2.0
print(a ** 2) # Exponentiation: 100

# String Operations
greeting = "Hello, "
name = "Rahul"
full_message = greeting + name  # Concatenation
print(full_message)  # Output: Hello, Rahul
print(name * 2)      # Repetition: RahulRahul

Use parentheses to control the order of operations, as Python follows PEMDAS (Parentheses, Exponents, Multiplication/Division, Addition/Subtraction).

Lists and Dictionaries

Python’s built-in data structures, like lists and dictionaries, are powerful for managing collections of data.

Lists: Ordered, mutable collections.


fruits = ["apple", "banana", "orange"]
print(fruits[0])       # Access first item: apple
fruits.append("grape") # Add item
print(fruits)          # Output: ['apple', 'banana', 'orange', 'grape']
fruits[1] = "mango"    # Modify item
print(fruits)          # Output: ['apple', 'mango', 'orange', 'grape']

Dictionaries: Key-value pairs for structured data.


student = {
    "name": "Rahul",
    "age": 25,
    "is_student": True
}
print(student["name"])        # Access value: Rahul
student["grade"] = "A"        # Add key-value pair
print(student)                # Output: {'name': 'Rahul', 'age': 25, 'is_student': True, 'grade': 'A'}
student["age"] = 26           # Update value
print(student["age"])         # Output: 26

Lists are great for ordered data, while dictionaries excel at mapping relationships. Use methods like len(), pop(), or keys() to manipulate them.

Conditions (If-Else)

Conditional statements control program flow based on conditions: 


age = 18
if age >= 18:
    print("You can vote!")
elif age >= 16:
    print("You can get a learner's permit.")
else:
    print("You cannot vote or drive.")

Output: You can vote!
Use elif for multiple conditions. Logical operators (and, or, not) can combine conditions, e.g., if age >= 18 and is_registered:.

Loops

Loops automate repetitive tasks. Python offers for and while loops:

For Loop: Iterates over a sequence.


for i in range(1, 6):
    print(f"Number: {i}")  # Output: Number: 1, Number: 2, ..., Number: 5

While Loop: Runs until a condition is false.


count = 1
while count <= 5:
    print(f"Count: {count}")
    count += 1  # Output: Count: 1, Count: 2, ..., Count: 5

Use break to exit a loop early or continue to skip an iteration. For example, if count == 3: continue skips printing 3.

Functions

Functions encapsulate reusable code. Define them with def: 


def greet(name, greeting="Hello"):
    return f"{greeting}, {name}!"
print(greet("Rahul"))          # Output: Hello, Rahul!
print(greet("Priya", "Hi"))    # Output: Hi, Priya!

Default parameters (e.g., greeting="Hello") make functions flexible. Use return to output values or None implicitly if no value is returned.

Error Handling

Python uses try and except to handle errors gracefully: 


try:
    number = int(input("Enter a number: "))
    print(f"Double: {number * 2}")
except ValueError:
    print("Please enter a valid number!")

This code catches invalid inputs (e.g., entering “abc” instead of a number). Use finally for cleanup or else for code that runs if no error occurs.

Practical Example: Simple Calculator

Let’s combine concepts to build a basic calculator: 


def calculator():
    try:
        num1 = float(input("Enter first number: "))
        op = input("Enter operator (+, -, *, /): ")
        num2 = float(input("Enter second number: "))
        
        if op == "+":
            result = num1 + num2
        elif op == "-":
            result = num1 - num2
        elif op == "*":
            result = num1 * num2
        elif op == "/":
            if num2 == 0:
                raise ValueError("Cannot divide by zero!")
            result = num1 / num2
        else:
            raise ValueError("Invalid operator!")
        
        print(f"Result: {result}")
    except ValueError as e:
        print(f"Error: {e}")

calculator()

This program uses functions, conditionals, error handling, and user input to perform calculations. Try running it to see how it handles valid and invalid inputs.

Conclusion

Python’s simplicity and power make it an excellent choice for beginners. This guide covered variables, data types, operations, lists, dictionaries, conditionals, loops, functions, and error handling. To deepen your skills:

  • Practice by building small projects, like a to-do list or number guessing game.
  • Explore Python’s standard library and modules like random or math.
  • Check out resources like the Python documentation or online platforms like Codecademy or freeCodeCamp.
Keep coding, and you’ll soon unlock Python’s full potential!

Practical Example: Simple Calculator

Concepts ko jodkar ek basic calculator banayein: 


def calculator():
    try:
        num1 = float(input("Pehla number daalein: "))
        op = input("Operator daalein (+, -, *, /): ")
        num2 = float(input("Doosra number daalein: "))
        
        if op == "+":
            result = num1 + num2
        elif op == "-":
            result = num1 - num2
        elif op == "*":
            result = num1 * num2
        elif op == "/":
            if num2 == 0:
                raise ValueError("Zero se divide nahi kar sakte!")
            result = num1 / num2
        else:
            raise ValueError("Galat operator!")
        
        print(f"Parinaam: {result}")
    except ValueError as e:
        print(f"Error: {e}")

calculator()
